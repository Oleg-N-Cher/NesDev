7. Using a mouse - mouse.h

Some target machines support a mouse. Mouse support is currently available for the following targets:

  • apple2
  • apple2enh
  • atari
  • c64
  • c128
  • cbm510

The available functions are declared in mouse.h To help writing portable code, the header file will define the symbol __MOUSE__ in systems that support a mouse.
