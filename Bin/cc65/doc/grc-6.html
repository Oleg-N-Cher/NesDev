6. Bugs and feedback

This is the first release of grc, and it contains bugs, for sure! I am aware of them; I know that the parser is weak, and if you don't follow the grammar rules strictly, then everything will crash. However, if you find an interesting bug, mail me. :-) Mail me also for help with writing your .grc file correctly if you have problems with it. I would appreciate comments also, and help on this file because I am sure that it can be written better.
