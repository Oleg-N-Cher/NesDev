5. Loadable drivers

5.1 Graphics drivers

No graphics drivers are currently available for the VIC20.

5.2 Extended memory drivers

No extended memory drivers are currently available for the VIC20.

5.3 Joystick drivers

vic20-stdjoy.joy

Supports one standard joystick connected to the joysticks port of the VIC20.

vic20-ptvjoy.joy

Driver for the Protovision 4-player adapter contributed by Groepaz. See http://www.protovision-online.de/hardw/hardwstart.htm for prices and building instructions. Up to three joysticks are supported.

5.4 Mouse drivers

No mouse drivers are currently available for the VIC20.

5.5 RS232 device drivers

No VIC1011 drivers are currently available for the VIC20.
