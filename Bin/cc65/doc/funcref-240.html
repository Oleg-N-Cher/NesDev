3.195 tgi_setpixel

Function

Plot a pixel on the drawpage with the current color.

Header

tgi.h

Declaration

void __fastcall__ tgi_setpixel (int x, int y);

Description

Plot a pixel on the drawpage with the current color.

Limits

  • The function is only available as fastcall function, so it may only be used in presence of a prototype.

Availability

cc65

See also

Other tgi functions.

Example

None.
